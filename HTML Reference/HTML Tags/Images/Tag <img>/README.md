# `<img>`

Изображение.

← [Изображения](../README.md) · [Справочник](../../README.md)

Пустой элемент (без закрывающего тега).

По умолчанию: `display: inline`.

```html
<img>
```

Атрибуты `src`, `alt`.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/img)
