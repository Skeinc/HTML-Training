# `<input>`

Поле ввода.

← [Формы и ввод](../README.md) · [Справочник](../../README.md)

Пустой элемент (без закрывающего тега).

По умолчанию: `display: inline`.

```html
<input>
```

Атрибут `type`.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/input)
