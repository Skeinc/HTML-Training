# `<button>`

Кнопка.

← [Формы и ввод](../README.md) · [Справочник](../../README.md)

Парный элемент.

По умолчанию: `display: inline`.

```html
<button>…</button>
```

Указывайте `type`.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/button)
