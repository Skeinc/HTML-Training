# `<datalist>`

Подсказки для input.

← [Формы и ввод](../README.md) · [Справочник](../../README.md)

Парный элемент.

Не отображается на странице (метаданные / скрытый шаблон).

```html
<datalist>…</datalist>
```

Связь через `list` на input.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/datalist)
