# `<source>`

Альтернативный медиа-источник.

← [Аудио и видео](../README.md) · [Справочник](../../README.md)

Пустой элемент (без закрывающего тега).

Не отображается на странице (метаданные / скрытый шаблон).

```html
<source>
```

В `<video>` или `<audio>`.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/source)
