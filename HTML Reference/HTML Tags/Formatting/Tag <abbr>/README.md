# `<abbr>`

Аббревиатура.

← [Форматирование](../README.md) · [Справочник](../../README.md)

Парный элемент.

По умолчанию: `display: inline`.

```html
<abbr>…</abbr>
```

`title` — расшифровка.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/abbr)
