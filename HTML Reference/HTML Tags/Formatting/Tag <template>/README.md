# `<template>`

Шаблон для клонирования в JS.

← [Форматирование](../README.md) · [Справочник](../../README.md)

Парный элемент.

Не отображается на странице (метаданные / скрытый шаблон).

```html
<template>…</template>
```

Не рендерится.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/template)
