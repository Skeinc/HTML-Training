# `<wbr>`

Возможный перенос строки.

← [Форматирование](../README.md) · [Справочник](../../README.md)

Пустой элемент (без закрывающего тега).

По умолчанию: `display: inline`.

```html
<wbr>
```

Пример — в [`index.html`](index.html).
