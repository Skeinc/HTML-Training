# Tag ins

Тег ``<ins>`` представляет собой элемент разметки, используемый для выделения вставленного текста, т.е. текста, который был добавлен к документу после его первоначальной публикации. Тег ``<ins>`` предназначен для выделения текста, который был вставлен или добавлен в документ после его первоначальной публикации. Обычно этот текст отображается подчеркнутым.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tag ins</title>
</head>
<body>
    <p>В новой версии сайта был <ins>добавлен</ins> новый раздел.</p>
</body>
</html>
```

## Атрибут datetime

Тег ``<ins>`` может содержать атрибут ``datetime``, который предоставляет машиночитаемую дату и время вставки. Это полезно для инструментов, которые могут анализировать историю изменений.

## Атрибут cite

Тег ``<ins>`` может содержать атрибут ``cite``, который указывает URL-адрес документа, который объясняет причину удаления/изменения текста.

## Использование в сочетании с тегом del

Часто тег ``<ins>`` используется вместе с тегом ``<del>``, который обозначает удаленный или измененный текст. Это помогает явно выделить разницу между удаленным и добавленным контентом.


## Примечания

- Обратите внимание на тег ``<del>`` для разметки вставленного текста.
- Тег ``<ins>`` поддерживает глобальные атрибуты в HTML.
- Тег ``<ins>`` поддерживает атрибуты событий в HTML.

## Заключение

Тег ``<ins>`` в HTML предоставляет простой способ выделения добавленного текста на вашем веб-сайте. Используйте его, чтобы подчеркнуть новые или измененные части контента и предоставить пользователям ясную информацию о внесенных изменениях.
