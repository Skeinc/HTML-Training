# Тег ul

Тег ``<ul>`` используется для создания неупорядоченных списков, то есть списков, где порядок элементов не имеет значения.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Тег ul</title>
</head>
<body>
    <ul>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
    </ul>
</body>
</html>
```

## Определение и использование

Тег ``<ul>`` определяет неупорядоченный (маркированный) список. Используйте этот ``<ul>`` тег вместе с тегом ``<li>`` для создания неупорядоченных списков.

## Примечания

- Используйте CSS для стилизации списков;
- Для упорядоченных списков используйте тег ``<ol>``;
- Тег ``<ul>`` поддерживает глобальные атрибуты в HTML;
- Тег ``<ul>`` поддерживает атрибуты событий в HTML.

## Настройки CSS по умолчанию

Большинство браузеров отображают ``<ul>`` элемент со следующими значениями по умолчанию:

```
ul {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}
```

## Заключение

Тег ``<ul>`` предоставляет удобный способ создания неупорядоченных списков на веб-страницах. Помните, что структурированный и читаемый код улучшает понимание и поддержку вашего проекта.
