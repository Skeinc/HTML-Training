# Заголовки `h1`–`h6`

Заголовки первого–шестого уровня.

← [Базовые теги](../README.md) · [Справочник](../../README.md)

Парный элемент.

По умолчанию: `display: block`.

```html
<h1>Заголовок</h1>
<h2>Подзаголовок</h2>
```

Чем больше номер — тем ниже уровень.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/Heading_Elements)
