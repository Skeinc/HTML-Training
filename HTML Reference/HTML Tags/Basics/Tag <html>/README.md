# `<html>`

Корневой элемент документа.

← [Базовые теги](../README.md) · [Справочник](../../README.md)

Парный элемент.

По умолчанию: `display: block`.

```html
<html>…</html>
```

Атрибут `lang` обязателен для a11y.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/html)
