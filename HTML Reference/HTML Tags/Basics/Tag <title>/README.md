# `<title>`

Заголовок вкладки браузера.

← [Базовые теги](../README.md) · [Справочник](../../README.md)

Парный элемент.

Не отображается на странице (метаданные / скрытый шаблон).

```html
<title>…</title>
```

Внутри `<head>`, обязателен.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/title)
