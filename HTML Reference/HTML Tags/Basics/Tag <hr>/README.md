# `<hr>`

Тематический разделитель.

← [Базовые теги](../README.md) · [Справочник](../../README.md)

Пустой элемент (без закрывающего тега).

По умолчанию: `display: block`.

```html
<hr>
```

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/hr)
