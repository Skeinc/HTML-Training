# `<style>`

Встроенные CSS-правила.

← [Стили и семантика](../README.md) · [Справочник](../../README.md)

Парный элемент.

Не отображается на странице (метаданные / скрытый шаблон).

```html
<style>…</style>
```

В `<head>`.

Пример — в [`index.html`](index.html).

- [MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/style)
