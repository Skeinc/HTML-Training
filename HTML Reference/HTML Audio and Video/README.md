# Audio и Video (справочник)

Нативное воспроизведение медиа без плагинов.

← [HTML Reference](../README.md)

Урок: [HTML Media](../../5.%20HTML%20Media/README.md)

| Тег | Назначение |
|-----|------------|
| `<video>` | Видео |
| `<audio>` | Аудио |
| `<source>` | Альтернативный файл/формат |
| `<track>` | Субтитры WebVTT |

```html
<video controls width="640">
    <source src="clip.webm" type="video/webm">
    <source src="clip.mp4" type="video/mp4">
</video>
```
