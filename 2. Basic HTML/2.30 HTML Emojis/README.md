# Эмодзи в HTML

Эмодзи — обычные символы **Unicode**, не картинки. Нужна кодировка UTF-8.

## Прямая вставка

```html
<head>
    <meta charset="UTF-8">
</head>
<body>
    <p>Привет! 👋 Работаем 🚀</p>
</body>
```

Без UTF-8 эмодзи отобразятся как «кракозябры».

## Числовые коды

Если символ не вставляется с клавиатуры:

```html
<p>&#128512; &#128156;</p>
<!-- 😀 и 💜 -->
```

Формат: `&#DEC;` — десятичный код Unicode. Для кодов выше 65535 в старых документах используют суррогатные пары — в современном HTML проще вставлять символ напрямую.

## Рекомендации

- В интерфейсах эмодзи — для тона, не вместо иконок с чётким смыслом (доступность)
- `alt` у декоративных картинок — пустой; эмодзи в тексте озвучиваются скринридером как есть
- Не путайте с HTML-сущностями вроде `&hearts;` (♥) — это другой символ, не цветное эмодзи
