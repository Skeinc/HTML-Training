## Таблица тегов

| Тег | Назначение |
|-----|------------|
| `<code>` | Фрагмент кода inline |
| `<pre>` | Блок с сохранением пробелов и переносов |
| `<kbd>` | Ввод с клавиатуры |
| `<samp>` | Вывод программы / терминала |
| `<var>` | Имя переменной |

## Примеры

```html
<p>Нажмите <kbd>Ctrl</kbd> + <kbd>S</kbd> для сохранения.</p>

<p>Ошибка: <samp>File not found</samp></p>

<p>Площадь: <var>S</var> = <var>a</var> × <var>b</var></p>

<pre><code>const sum = (a, b) => a + b;</code></pre>
```

`<code>` один не сохраняет отступы — для многострочного кода оборачивайте в `<pre>`.

## Символы `<` и `>`

В тексте внутри кода экранируйте или пишите в `<pre>`/`<code>` аккуратно. В обычном тексте: `&lt;` и `&gt;` — см. [Сущности](../2.28%20HTML%20Entities/README.md).
