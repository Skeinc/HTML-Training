# Цвета в HTML

Цвет задаётся в CSS — через атрибут `style`, `<style>` или внешний файл. В HTML доступны несколько форматов записи.

## Форматы

| Формат | Пример | Когда использовать |
|--------|--------|-------------------|
| Имя | `tomato`, `dodgerblue` | ~140 имен; быстро для прототипов |
| HEX | `#ff6347` | Самый частый в продакшене |
| RGB | `rgb(255, 99, 71)` | Когда удобно по каналам 0–255 |
| RGBA | `rgba(255, 99, 71, 0.5)` | + прозрачность (alpha 0–1) |
| HSL | `hsl(9, 100%, 64%)` | Удобно крутить оттенок/насыщенность |
| HSLA | `hsla(9, 100%, 64%, 0.5)` | HSL + прозрачность |

**HSL:** оттенок `0–360` (0=красный, 120=зелёный, 240=синий), насыщенность и светлота — в `%`.

## Примеры

```html
<p style="color: #333;">Текст</p>
<p style="background-color: rgb(240, 248, 255);">Фон</p>
<div style="border: 2px solid hsl(200, 80%, 50%);">Рамка</div>
```

Свойства: `color` (текст), `background-color`, `border-color` и др.
