## UTF-8

В HTML5 по умолчанию — **UTF-8**. Объявите явно в первых строках `<head>`:

```html
<meta charset="UTF-8">
```

UTF-8 кодирует любые символы Unicode (латиница, кириллица, иероглифы, эмодзи). Один стандарт на весь проект.

Файл `.html` тоже сохраняйте в UTF-8 в редакторе.

## История (кратко)

| Набор | Эпоха | Заметка |
|-------|-------|---------|
| ASCII | 128 символов | Только английский |
| Windows-1252 / ISO-8859-1 | HTML4 | Западная Европа |
| UTF-8 | HTML5, сейчас | Универсальный выбор |

Старые `charset=windows-1251` встречаются в легаси, но новые проекты — только UTF-8.

## Практика

- Первая строка метаданных в `<head>` — `charset`
- HTTP-заголовок `Content-Type: text/html; charset=utf-8` на сервере согласован с meta
- Если видите `` — файл или заголовок в неверной кодировке
