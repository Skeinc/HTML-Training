## Таблица тегов

| Тег | Смысл | Визуально (по умолчанию) |
|-----|-------|--------------------------|
| `<strong>` | Важность | жирный |
| `<b>` | Стилистическое выделение без особой важности | жирный |
| `<em>` | Акцент | курсив |
| `<i>` | Альтернативный тон (термин, иностр. слово) | курсив |
| `<mark>` | Подсветка/выделение в контексте | фон |
| `<small>` | Второстепенный текст (сноски, дисклеймер) | мельче |
| `<del>` | Удалённый текст | зачёркнутый |
| `<ins>` | Добавленный текст | подчёркнутый |
| `<s>` | Неактуальное (устаревшая цена и т.п.) | зачёркнутый |
| `<sub>` / `<sup>` | Нижний / верхний индекс | H₂O, x² |
| `<u>` | Подчёркнутый (редко; путается со ссылками) | подчёркнутый |

```html
<p><strong>Важно:</strong> сдайте работу до <em>пятницы</em>.</p>
<p>Цена: <del>1000 ₽</del> <ins>800 ₽</ins></p>
<p>H<sub>2</sub>O, E=mc<sup>2</sup></p>
```

## Важно

- Для **важности** — `<strong>`, для **акцента** — `<em>`, не наоборот
- Внешний вид всегда можно переопределить CSS; смысл тега остаётся для скринридеров и SEO
- `<b>` и `<i>` — когда нужен только визуальный эффект без семантики
