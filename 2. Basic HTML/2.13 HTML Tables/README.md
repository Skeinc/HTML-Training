## Базовая структура

```html
<table>
    <tr>
        <th>Компания</th>
        <th>Контакт</th>
        <th>Страна</th>
    </tr>
    <tr>
        <td>Sony</td>
        <td>Maria Anders</td>
        <td>Germany</td>
    </tr>
    <tr>
        <td>Acme</td>
        <td>Ivan Petrov</td>
        <td>Russia</td>
    </tr>
</table>
```

| Тег | Назначение |
|-----|------------|
| `<table>` | Таблица |
| `<tr>` | Строка |
| `<th>` | Ячейка-заголовок (жирная, по центру по умолчанию) |
| `<td>` | Ячейка с данными |

## Заголовки и подпись

```html
<table>
    <caption>Сотрудники отдела</caption>
    <tr>
        <th>Имя</th>
        <th>Должность</th>
    </tr>
    <tr>
        <td>Анна</td>
        <td>Разработчик</td>
    </tr>
</table>
```

`<caption>` — заголовок таблицы, сразу после `<table>`.

## Объединение ячеек

```html
<tr>
    <th colspan="2">Имя</th>
    <th>Возраст</th>
</tr>
<tr>
    <td colspan="2">Иван Иванов</td>
    <td>28</td>
</tr>
<tr>
    <td rowspan="2">Телефон</td>
    <td>555-1234</td>
    <td></td>
</tr>
```

| Атрибут | Эффект |
|---------|--------|
| `colspan="N"` | Ячейка на N столбцов |
| `rowspan="N"` | Ячейка на N строк |

## Секции таблицы

Для больших таблиц — семантические группы:

```html
<table>
    <thead>
        <tr><th>Месяц</th><th>Продажи</th></tr>
    </thead>
    <tbody>
        <tr><td>Январь</td><td>100</td></tr>
    </tbody>
    <tfoot>
        <tr><td>Итого</td><td>100</td></tr>
    </tfoot>
</table>
```

`<colgroup>` / `<col>` — стилизация столбцов (ограниченный набор CSS-свойств).

## Стили

Границы, отступы и зебра — через CSS:

```css
table { border-collapse: collapse; width: 100%; }
th, td { border: 1px solid #ccc; padding: 8px; }
tr:nth-child(even) { background: #f9f9f9; }
```

Размеры лучше задавать в CSS, не атрибутами `width`/`height` на каждой ячейке.

## Доступность

- Используйте `<th>` с `scope="col"` или `scope="row"` для заголовков
- Для сложных таблиц — `headers` и `id` на ячейках
- Не используйте таблицы для колоночного макета — [Flex/Grid](../2.23%20HTML%20Layout/README.md)
