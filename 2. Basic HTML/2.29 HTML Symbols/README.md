# Специальные символы в HTML

Математика, валюта, стрелки, греческие буквы — через **именованные сущности** или **Unicode** в UTF-8.

← [Базовый HTML](../README.md) · [Сущности](../2.28%20HTML%20Entities/README.md)

- [Как вставлять](#как-вставлять)
- [Примеры](#примеры)

При `<meta charset="UTF-8">` символы вроде `©`, `→`, `α` можно вставлять **напрямую** с клавиатуры или буфера. Сущности удобны, когда символа нет на клавиатуре.

## Как вставлять

```html
<!-- именованная сущность -->
<p>&forall; x &isin; &#8477;</p>

<!-- десятичный номер Unicode -->
<p>&#8594; стрелка вправо</p>

<!-- символ UTF-8 напрямую -->
<p>© → α β γ</p>
```

## Примеры

**Математика:**

| Символ | Сущность |
|--------|----------|
| ∀ | `&forall;` |
| ∂ | `&part;` |
| ∑ | `&sum;` |
| ∈ | `&isin;` |
| ∅ | `&empty;` |

**Греческие буквы:**

| | | |
|---|---|---|
| Α `&Alpha;` | β прямой Unicode | π `&pi;` |

**Прочее:**

| Символ | Сущность |
|--------|----------|
| © | `&copy;` |
| ® | `&reg;` |
| ™ | `&trade;` |
| ← → ↑ ↓ | `&larr;` `&rarr;` `&uarr;` `&darr;` |
| ♠ ♣ ♥ ♦ | `&spades;` `&clubs;` `&hearts;` `&diams;` |

Полные таблицы — [MDN: Entity reference](https://developer.mozilla.org/en-US/docs/Glossary/Entity).
