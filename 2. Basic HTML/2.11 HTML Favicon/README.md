## Подключение

В `<head>`:

```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/icon.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
```

| Файл | Назначение |
|------|------------|
| `favicon.ico` | Классический формат, 16×16 / 32×32 |
| `.svg` | Масштабируется, один файл на все размеры |
| `apple-touch-icon.png` | Иконка при «Добавить на экран» iOS (обычно 180×180) |

Можно положить `favicon.ico` в корень сайта — браузеры найдут и без `<link>`, но явное подключение надёжнее.

## Рекомендации

- Простой узнаваемый силуэт — на 16×16 детали не видны
- Несколько форматов: ICO/PNG + SVG
- Путь абсолютный от корня (`/favicon.ico`) — не ломается на вложенных страницах
