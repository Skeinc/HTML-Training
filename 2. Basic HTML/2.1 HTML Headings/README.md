# Заголовки в HTML

Теги `h1`–`h6` задают **иерархию** заголовков. Размер шрифта задаёт CSS, не номер тега.

## Уровни

| Тег | Обычное назначение |
|-----|-------------------|
| `h1` | Главный заголовок страницы (обычно один) |
| `h2` | Крупные разделы |
| `h3`–`h6` | Подразделы; чем **больше** номер, тем **ниже** уровень |

```html
<h1>Курс HTML</h1>
<h2>Глава 1. Основы</h2>
<h3>1.1 Теги</h3>
```

## Правила

1. Соблюдайте порядок: не прыгайте с `h2` сразу на `h4` без причины
2. Один `h1` на логическую страницу — хорошая практика для доступности и SEO
3. Не используйте заголовки только ради крупного текста — для этого есть CSS
4. Заголовок должен отражать смысл блока, а не «красиво выглядеть»
