## Относительные пути

От **текущего файла** или от **корня сайта**:

| Запись | Значение |
|--------|----------|
| `photo.jpg` | Файл в той же папке, что и HTML |
| `images/photo.jpg` | Подпапка `images/` |
| `../photo.jpg` | На уровень выше |
| `/images/photo.jpg` | От корня сайта (не путать с диском `C:\`) |

```html
<img src="images/logo.png" alt="Логотип">
<link rel="stylesheet" href="../css/main.css">
<a href="/about.html">О нас</a>
```

## Абсолютные URL

Полный адрес — для внешних ресурсов:

```html
<img src="https://cdn.example.com/banner.jpg" alt="Баннер">
<script src="https://cdn.example.com/lib.js"></script>
```

## Рекомендации

- Для **своих** файлов — относительные пути или от корня (`/css/style.css`)
- Имена файлов — **строчными** буквами (Linux-серверы чувствительны к регистру)
- При переносе домена относительные ссылки не ломаются
- `..` в пути — только внутри проекта, не выходите «наружу» сайта без нужды
