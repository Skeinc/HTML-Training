# Прогресс и шкала (`<progress>`, `<meter>`)

Два похожих тега для отображения числового состояния — с разным смыслом.

## `<progress>`

**Ход выполнения задачи** — загрузка, установка, прохождение теста. Значение может расти со временем.

```html
<label for="upload">Загрузка файла</label>
<progress id="upload" value="45" max="100">45%</progress>
```

| Атрибут | Назначение |
|---------|------------|
| `value` | Текущее значение |
| `max` | Максимум (по умолчанию 1.0) |

Без `value` — **неопределённый** прогресс (анимация «идёт работа»).

## `<meter>`

**Измерение в известном диапазоне** — уровень диска, оценка, температура. Не «процесс», а «показатель».

```html
<label for="disk">Место на диске</label>
<meter id="disk" value="7" min="0" max="10" low="3" high="8" optimum="2">7 из 10</meter>
```

| Атрибут | Назначение |
|---------|------------|
| `value` | Текущее значение (обязателен) |
| `min` / `max` | Диапазон |
| `low` / `high` | Зоны «низко» / «высоко» |
| `optimum` | Оптимальное значение (для цвета в браузере) |

Текст внутри тега — fallback для старых браузеров и скринридеров.

## Сравнение

| | `<progress>` | `<meter>` |
|---|--------------|-----------|
| Смысл | Процесс идёт | Статическое измерение |
| Пример | Загрузка 60% | Рейтинг 4.2 из 5 |
| `value` без числа | Допустим (неопределённый) | Нет — всегда число |

Для ползунка ввода в форме используйте `<input type="range">` — [типы input](../../3.%20HTML%20Forms/3.4%20HTML%20Input%20Types/README.md).

Пример — в [`index.html`](index.html).
