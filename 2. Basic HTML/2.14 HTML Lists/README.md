## Маркированный `ul`

Порядок не важен:

```html
<ul>
    <li>Кофе</li>
    <li>Чай</li>
    <li>Молоко</li>
</ul>
```

Маркер меняется в CSS: `list-style-type: disc | circle | square | none`.

## Нумерованный `ol`

Порядок важен:

```html
<ol>
    <li>Включить духовку</li>
    <li>Смешать ингредиенты</li>
    <li>Выпекать 30 минут</li>
</ol>
```

| Атрибут | Назначение |
|---------|------------|
| `type="1\|A\|a\|I\|i"` | Стиль нумерации |
| `start="5"` | Начать с числа 5 |
| `reversed` | Обратный порядок |

## Список определений `dl`

Термин + описание:

```html
<dl>
    <dt>HTML</dt>
    <dd>Язык разметки гипертекста</dd>
    <dt>CSS</dt>
    <dd>Язык оформления</dd>
</dl>
```

## Вложенность

Списки вкладывают в `<li>`, не «между» `<ul>`:

```html
<ul>
    <li>Напитки
        <ul>
            <li>Кофе</li>
            <li>Чай</li>
        </ul>
    </li>
    <li>Еда</li>
</ul>
```

Навигационные меню часто строят на `<ul>`/`<li>` внутри `<nav>` — см. [Семантика](../2.26%20HTML%20Semantics/README.md).
