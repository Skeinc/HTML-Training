# Комментарии в HTML

Комментарии видны в исходнике, но **не отображаются** на странице и не влияют на рендер.

← [Базовый HTML](../README.md)

- [Синтаксис](#синтаксис)
- [Зачем использовать](#зачем-использовать)
- [Ограничения](#ограничения)

## Синтаксис

```html
<!-- однострочный комментарий -->

<!--
    многострочный
    комментарий
-->

<p>Видимый текст</p>
<!-- <p>Этот абзац временно отключён</p> -->
```

## Зачем использовать

- Пояснить неочевидную разметку для команды
- Временно отключить блок при отладке
- Отметить TODO или секции в шаблоне

## Ограничения

- **Нельзя** вкладывать `<!--` внутрь другого комментария
- Комментарии **не безопасны** для секретов — их видит любой в «Просмотре кода»
- Не комментируйте внутри атрибута: `<p title="<!-- нет -->">` — сломает разметку
