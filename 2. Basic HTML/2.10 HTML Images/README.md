# Изображения в HTML

Изображения улучшают визуальный опыт пользователей, делая веб-страницы более привлекательными и информативными. Изображения могут улучшить дизайн и внешний вид веб-страницы.

```
<img src="./image.jpg" alt="leaves in autumn">
```

## Синтаксис HTML-изображений

Тег HTML ``<img>`` используется для встраивания изображения на веб-странице. Изображения технически не вставляются на веб-страницу, изображения связаны с сетью страницы. Тег ``<img>`` создает холдинг место для указанного изображения. Тег ``<img>`` пустой, содержит только атрибуты и не имеет закрывающего тега.

Тег ``<img>`` имеет два обязательных атрибута:

- **src:** указывает путь к изображению.
- **alt:** указывает альтернативный текст для изображения.

Тег ``<img>`` имеет следующий синтаксис:

```
<img src="url" alt="alternativetext">
```

## Атрибут src

Обязательный атрибут src указывает путь (URL) к изображению.

**Примечание:** Когда веб-страница загружается, при этом браузер, в этот момент, получает изображение с веб-сервера и вставляет его на страницу. Поэтому убедитесь, что изображение действительно остается на том же месте относительно веб-страницы, иначе ваши посетители увидят значок неработающей ссылки. Сломанный значок ссылки и текст ``alt`` отображаются, если браузер не может найти изображение.

```
<img src="./image.jpg">
```

## Атрибут alt

Обязательный атрибут ``alt`` предоставляет альтернативный текст для изображения, если пользователь почему-то не может просмотреть изображение. Значение атрибута ``alt`` должно описывать изображение. Если браузер не может найти изображение, он отобразит значение ``alt`` атрибута:

```
<img src="./image.jpg" alt="leaves in autumn">
```

## Размер изображения

Вы можете использовать атрибут ``style``, чтобы указать ширину и высоту изображения, или можно использовать атрибуты ``width`` и ``height``:

```
<img src="./image.jpg" alt="leaves in autumn" width="300" height="200">
```

Атрибуты ``width`` и ``height`` всегда определяют ширину и высоту изображения в пикселях.

**Примечание:** Всегда указывайте ширину и высоту изображения. Если ширина и высота не указаны, веб-страница может мерцать во время загрузки изображения.

## Ширина и высота, или стиль?

Атрибуты ``width``, ``height`` и ``style`` являются допустимыми в HTML. Однако лучше использовать атрибут ``style``. Это предотвращает изменение таблиц стилей:

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Images</title>
    <style>
        .image {
            width: 400px;
            height: 400px;
            object-fit: cover;
        }
    </style>
</head>
<body>
    <h2>Width and Height or Style?</h2>
    <img src="./image.jpg" alt="leaves in autumn" class="image">
</body>
</html>
```

## Изображения в другой папке

Если у вас есть изображения в подпапке, вы должны включить имя папки в атрибуте ``src``:

```
<img src="/images/image.jpg" alt="leaves in autumn">
```

## Изображения на другом сервере/веб-сайте

Некоторые веб-сайты указывают на изображение на другом сервере. Чтобы указать изображение на другом сервере, необходимо указать абсолютный (полный) URL в атрибуте ``src``:

```
<img src="https://google.com/photos/image.jpeg" alt="leaves in autumn">
```

**Примечание относительное внешних изображений:** Внешние изображения могут находиться под Авторские права. Если вы не получили разрешения на его использование, возможно, вы нарушаете законы об авторском праве. Кроме того, вы не можете управлять внешними изображениями, они могут внезапно быть удалены или изменены.

## Анимированные изображения

HTML позволяет использовать анимированные GIF-изображения:

```
<img src="./image.gif" alt="leaves in autumn">
```

## Изображение как ссылка

Чтобы использовать изображение в качестве ссылки, поместите тег ``<img>`` внутри тега ``<a>``:

```
<a href="./README.md">
    <img src="./image.jpg" alt="leaves in autumn">
</a>
```

## Плавающее изображение

Используйте свойство CSS ``float``, чтобы изображение располагалось справа или слева от текста:

```
<p><img src="./image.jpg" alt="leaves in autumn" style="float: right;">The image will float to the right of the text.</p>
```

## Ленивая загрузка

Атрибут ``loading="lazy"`` позволяет браузеру загружать изображения только при приближении к ним во время прокрутки страницы:

```
<img src="./image.jpg" alt="leaves in autumn" loading="lazy">
```

## Фигурные изображения (SVG)

SVG (Scalable Vector Graphics) - это формат изображений в виде XML-кода, который позволяет создавать масштабируемые векторные изображения:

```
<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red">
</svg>
```

## Распространенные форматы изображений

Вот наиболее распространенные типы файлов изображений, которые поддерживаются всеми браузерами:

- **APNG:** Анимированные портативные сетевые графики (.apng)
- **GIF:** Графический формат обмена (.gif)
- **ICO:** Иконки от Microsoft (.ico, .cur)
- **JPEG:** Изображения в формате JPEG (.jpg, .jpeg, .jfif, .pjpeg, .pjp)
- **PNG:** Портативная графика в сети (.png)
- **SVG:** Масштабируемая векторная графика (.svg)

**Примечание:** Загрузка больших изображений требует времени и может замедлить работу веб-страницы. Используйте изображения осторожно

## Карты изображений

С помощью карт изображений HTML вы можете создавать на изображении интерактивные области. Тег HTML ``<map>`` определяет карту изображения. Карта изображений - это изображение с кликабельной областью. Области определяются одним или несколькими тегами ``<area>``:

```
<img src="./image.jpg" alt="leaves in autumn" usemap="#workmap">

<map name="workmap">
    <area shape="rect" coords="0, 0, 100, 100" href="#area-first" alt="area first">
    <area shape="rect" coords="200, 200, 300, 300" href="#area-second" alt="area second">
    <area shape="rect" coords="100, 200, 100" href="#area-third" alt="area third">
    <area shape="poly" coords="50,100,150,200,250,300,350,400,450,500" href="#area-fourth" alt="area fouth">
</map>
```

### Как это работает?

Идея карты изображений заключается в том, что вы должны иметь возможность выполнять различные действия в зависимости от того, где на изображении вы нажимаете. Чтобы создать карту изображений, вам понадобится изображение и некоторый HTML-код, описывающий интерактивные области.

### Изображение

Изображение вставляется с помощью тега ``<img>``. Единственное отличие от других изображений заключается в том, что вам необходимо добавить атрибут ``<usemap>``.

Значение ``usemap`` начинается с хеш-тега ``#``, за которым следует имя карты изображений, используется для создания связи между изображением и картой изображения.

### Создать карту изображений

Добавьте элемент ``<map>``. Элмент ``<map>`` используется для создания карты изображения и привязывается к изображению с помощью атрибута ``name``. Атрибут ``name`` должен иметь то же значение, что и атрибут ``usemap``.

### Области

Добавьте кликабельные области. Интерактивная область определяется с помощью элемента ``<area>``.

Вы должны определить форму кликабельной области и можете выбрать один из этих вариантов:

- **rect** - определяет прямоугольную область
- **circle** - определяет круговую область
- **poly** - определяет полигональную область
- **default** - определяет весь регион

Вы также должны определить некоторые координаты, чтобы имет возможность разместить кликабельную область на изображение.

#### Форма прямоугольник

Координаты для ``shape="rect"`` представлены парами: одна для оси X и одна для оси Y.

#### Форма круг

Чтобы добавить область круга, сначала найдите координаты центра круга, затем укажите радиус круга.

#### Форма полигональная

``shape="poly"`` содержит несколько координат точек, что создает форму образованную прямыми линиями (многоугольник). С его помощью можно создать любую форму.

### Карта изображений и JavaScript

Кликабельная область также может вызвать функцию JavaScript. Добавьте событие ``click`` в элементе ``<area>`` для выполнения функции JavaScript.

## Фоновые изображения

Фоновое изображение можно указать практически для любого элемента HTML. Чтобы добавить фоновое изображение в элемент HTML, используйте атрибут HTML ``style`` и свойство CSS ``background-image``:

```
<div style="background-image: url('./image.jpg')"></div>
```

Если вы хотите, чтобы фоновое изображение закрывало весь элемент, вы можете установить для свойства ``background-size`` значение ``cover``. Кроме того, чтобы убедиться, что весь элемент всегда покрыт, установите для свойства ``background-attachment`` значение ``fixed``:

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Images</title>
    <style>
        .image-background {
            background-image: url('./image.jpg');
            background-repeat: no-repeat;
            background-attachment: fixed;
            background-size: cover;
        }
    </style>
</head>
<body>
    <h1>HTML Background Images</h1>
    <div style="width: 100%; height: 400px;" class="image-background">

    </div>
</body>
</html>
```

## HTML элемент picture

Элемент HTML ``<picture>`` позволяет вам показывать разные картинки для разных устройств или размеров экрана. Элемент ``<picture>`` дает веб-сайту разработчикам больше гибкости в указание ресурсов изображения. Элемент ``<picture>`` содержит один или больше элементов ``<source>``, каждый из которых ссылается к различным изображениям через ``srcset`` атрибут. Таким образом, браузер может выбрать изображение, которое лучше всего соответствует текущему виду и/или устройству. Каждый элемент ``<source>`` имеет атрибут ``media``, определяющий, когда изображение является наиболее подходящим:

```
<picture>
    <source media="(max-width: 650px)" srcset="./image.png">
    <source media="(max-width: 400px)" srcset="./image.svg">
    <img src="./image.jpg" alt="leaves in autumn">
</picture>
```

**Примечание:** Всегда указывайте элемент ``<img>`` в качестве последнего дочернего элемента. Элемент ``<img>`` используется браузерами, которые не поддерживают элемент ``<picture>`` или ни один из тегов ``<source>`` не соответствует.

### Когда использовать элемент изображения

У элемента ``<picture>`` есть две основные цели:

1. **Пропускная способность**

    Если у вас маленький экран или устройство, не обязательно загружать большой файл изображения. Браузер будет использовать первый ``<source>`` элемент с совпадающими значениями атрибутов и игнорировать любое из следующего.

2. **Поддержка форматов**

    Некоторые браузеры или устройства могут поддерживать не все форматы изображений. С помощью ``<picture>``, вы можете добавить изображения всех форматов, и браузер будет использовать первый распознанный формат и игнорировать любые из следующих элементов.

**Примечание:** Браузер будет использовать первый элемент ``<source>`` и игнорировать последующие элементы ``<source>``.

## Заключение

Вставка изображений в HTML - важный аспект веб-разработки, который способствует созданию привлекательного и информативного контента. Используя тег ``<img>`` и соответствующие атрибуты, вы можете эффективно управлять изображениями на ваших веб-страницах.
