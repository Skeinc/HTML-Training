## Семантика vs `div`

| Несемантические | Семантические |
|-----------------|---------------|
| `div`, `span` | `header`, `nav`, `main`, `article`, `form`, `table`… |

`div`/`span` — когда **нет** подходящего смыслового тега.

## Теги макета

| Тег | Назначение |
|-----|------------|
| `header` | Вводная часть страницы или секции |
| `nav` | Навигационные ссылки |
| `main` | Основное содержимое (один на страницу) |
| `article` | Самостоятельный материал (пост, карточка товара) |
| `section` | Тематическая группа с заголовком |
| `aside` | Боковой/дополнительный контент |
| `footer` | Подвал страницы или секции |
| `figure` / `figcaption` | Иллюстрация с подписью |
| `details` / `summary` | Раскрывающийся блок |
| `dialog` | Модальное окно — [урок](../2.34%20HTML%20Dialog/README.md) |
| `time` | Дата/время (`datetime`) |

## `article` и `section`

- **`article`** — можно вынести в RSS/поделиться отдельно (пост блога, комментарий)
- **`section`** — логический раздел **внутри** страницы или `article`

Вложение допустимо в обе стороны по смыслу: блог (`article` внутри `section`) или глава (`section` внутри `article`).

## Пример страницы

```html
<body>
    <header>
        <h1>Мой блог</h1>
        <nav>
            <a href="/">Главная</a>
            <a href="/about">О сайте</a>
        </nav>
    </header>

    <main>
        <article>
            <header>
                <h2>Заголовок поста</h2>
                <time datetime="2026-06-11">11 июня 2026</time>
            </header>
            <p>Текст статьи…</p>
        </article>
    </main>

    <aside>
        <h2>Популярное</h2>
        <ul>…</ul>
    </aside>

    <footer>
        <p>© 2026</p>
    </footer>
</body>
```

На одной странице может быть несколько `header`/`footer` — внутри `article` или `section`.
