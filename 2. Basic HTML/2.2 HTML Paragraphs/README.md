## Абзац `p`

```html
<p>Первый абзац.</p>
<p>Второй абзац — браузер добавит отступы между ними.</p>
```

Внутри `<p>` можно вкладывать inline-элементы (`<a>`, `<strong>`, `<img>`…), но не другие блочные (`<div>`, `<p>`).

**Пробелы и переносы в исходнике** браузер схлопывает в один пробел. Чтобы разбить строку внутри абзаца, используйте `<br>`, а не несколько Enter в HTML.

## Перенос и разделитель

| Тег | Назначение |
|-----|------------|
| `<br>` | Перенос строки внутри абзаца (пустой элемент) |
| `<hr>` | Тематический разделитель (горизонтальная линия) |

```html
<p>Строка одна<br>Строка два</p>
<hr>
<p>Следующий блок контента.</p>
```

## Предформатированный текст

`<pre>` сохраняет пробелы и переносы — удобно для кода и ASCII-арта:

```html
<pre>
function hello() {
    return 'Hi';
}
</pre>
```

Для фрагментов кода в тексте чаще используют `<code>` внутри `<pre>` — см. [Computercode](../2.25%20HTML%20Computercode/README.md).
