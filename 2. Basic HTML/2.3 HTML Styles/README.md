# Инлайн-стили в HTML

Атрибут `style` задаёт CSS **прямо на элементе**. Удобно для прототипов; в проектах лучше выносить стили в файл.

- [Дальше](#дальше)

## Синтаксис

```html
<p style="color: red; font-size: 18px;">Красный текст</p>
<p style="background: #eee; padding: 8px;">Блок с фоном</p>
```

Несколько свойств разделяются точкой с запятой. Имена свойств — как в CSS (`background-color`, не `backgroundColor`).

## Когда уместно

| ✅ | ❌ |
|----|-----|
| Быстрый прототип, одноразовое оформление | Повторяющиеся стили на многих элементах |
| Динамические стили из JavaScript | Вся вёрстка страницы |

Для системного оформления используйте [CSS в HTML](../2.8%20HTML%20Styles%20CSS/README.md): `<style>` в `<head>` или внешний `.css`.
