## Семантический каркас

```html
<body>
    <header>
        <h1>Сайт</h1>
        <nav><!-- ссылки --></nav>
    </header>

    <main>
        <article>
            <h2>Статья</h2>
            <p>Контент…</p>
        </article>
        <aside>Боковая колонка</aside>
    </main>

    <footer>© 2026</footer>
</body>
```

| Тег | Роль |
|-----|------|
| `header` | Шапка страницы или секции |
| `nav` | Блок навигации |
| `main` | Основной контент (один на страницу) |
| `article` | Самостоятельный материал |
| `aside` | Дополнение к основному |
| `footer` | Подвал |
| `section` | Тематическая группа с заголовком |

Подробнее — [Семантика](../2.26%20HTML%20Semantics/README.md).

## Способы вёрстки колонок

| Метод | Статус |
|-------|--------|
| **Flexbox** | Колонки в ряд, выравнивание, адаптив |
| **CSS Grid** | Сетка строк и столбцов |
| **Float** | Устарело для макета; только исключения |
| **Таблицы** | Только для табличных **данных** |
| **CSS-фреймворки** | Bootstrap и др. — готовые сетки |

Минимальный Flex-макет «шапка + две колонки + подвал»:

```css
.layout { display: flex; flex-direction: column; min-height: 100vh; }
.content { display: flex; flex: 1; gap: 1rem; }
main { flex: 1; }
aside { width: 240px; }
```

Полные примеры float/flex/grid — в [`index.html`](index.html).
