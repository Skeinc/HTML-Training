# Заголовок страницы (`<title>`)

Текст на вкладке браузера, в закладках и часто в сниппете поисковика.

## Синтаксис

```html
<head>
    <title>Курс HTML — Глава 2. Ссылки</title>
</head>
```

`<title>` **обязателен** в валидном HTML-документе.

## Рекомендации

| ✅ | ❌ |
|----|-----|
| Уникальный title на каждой странице | Одинаковый «Главная» везде |
| Кратко и по сути (≈50–60 символов для сниппета) | Переспам ключевыми словами |
| Сначала суть, потом бренд: `Таблицы — HTML-курс` | Длинные бессмысленные строки |

Title дополняет, но не заменяет заголовки `h1`–`h6` на странице.

## Динамическое изменение

В SPA и уведомлениях title меняют из JavaScript:

```javascript
document.title = 'Новое сообщение (3)';
```
