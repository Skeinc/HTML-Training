## Различия

| | Блочный (block) | Строчный (inline) |
|---|-----------------|-------------------|
| Новая строка | Да | Нет |
| Ширина | На всю доступную | По содержимому |
| Примеры | `p`, `div`, `h1`–`h6`, `ul`, `section` | `a`, `span`, `strong`, `img`, `code` |

Современные элементы (`flex`, `grid`, `inline-block`) меняют поведение через CSS.

## Примеры

```html
<p>Абзац — блочный элемент.</p>
<p>Второй абзац с <strong>важным</strong> словом inline.</p>
```

## Правило вложенности

**Inline не должен содержать block** — разметка будет невалидной:

```html
<!-- неверно -->
<span><p>текст</p></span>

<!-- верно -->
<p><span>текст</span></p>
```

Исключения в HTML5: некоторые элементы с «гибридным» поведением (`button`, `a` с блочным содержимым в HTML5).

## `div` и `span`

| Тег | Тип | Назначение |
|-----|-----|------------|
| `<div>` | block | Универсальный контейнер без семантики |
| `<span>` | inline | Универсальная обёртка фрагмента текста |

Для смысла предпочитайте семантику: [Семантика](../2.26%20HTML%20Semantics/README.md).
