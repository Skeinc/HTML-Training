## Синтаксис

```html
<a href="https://example.com">Текст ссылки</a>
<a href="./about.html">О нас</a>
<a href="mailto:hello@example.com">Написать</a>
<a href="tel:+79001234567">Позвонить</a>
```

Ссылкой может быть не только текст — оберните `<img>`, кнопку (лучше стилизованный `<a>`) и др.

## Абсолютные и относительные URL

| Тип | Пример | Когда |
|-----|--------|-------|
| Абсолютный | `https://example.com/page` | Внешний сайт |
| Относительный | `./page.html`, `../img/logo.png` | Файлы своего проекта |

## Атрибуты

| Атрибут | Назначение |
|---------|------------|
| `href` | Куда ведёт ссылка (обязателен) |
| `target="_blank"` | Открыть в новой вкладке |
| `rel="noopener noreferrer"` | С `_blank` — безопасность (нет доступа к `window.opener`) |
| `title` | Подсказка при наведении |
| `download` | Скачать файл вместо перехода |

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
    Внешний сайт
</a>
```

Состояния ссылки стилизуются в CSS: `:link`, `:visited`, `:hover`, `:active`.

## Якоря

Ссылка на фрагмент той же страницы — через `#id`:

```html
<h2 id="раздел-2">Раздел 2</h2>
<a href="#раздел-2">Перейти к разделу 2</a>
```

`id` должен быть уникален на странице — см. [HTML Id](../2.18%20HTML%20Id/README.md).
