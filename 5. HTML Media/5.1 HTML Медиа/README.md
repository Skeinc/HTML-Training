## Форматы

Указывайте **несколько** `<source>` — браузер выберет первый поддерживаемый.

| Тип | Рекомендуемые форматы |
|-----|----------------------|
| Видео | MP4 (H.264), WebM (VP9/AV1) |
| Аудио | MP3, AAC (в MP4), Ogg Vorbis, Opus (в WebM) |

Устарели для веба: Flash, AVI, WMV как основной формат на странице.

MIME в `type` помогает браузеру:

```html
<source src="clip.webm" type="video/webm">
<source src="clip.mp4" type="video/mp4">
```

## Как встраивать

| Контент | Элемент |
|---------|---------|
| Свой файл | `<video>`, `<audio>` |
| YouTube, Vimeo | `<iframe>` с embed-URL |
| Фоновое видео | `<video autoplay muted loop playsinline>` + CSS |

Всегда давайте **управление** (`controls`) или явную кнопку play — автозвук без действия пользователя браузеры блокируют.
