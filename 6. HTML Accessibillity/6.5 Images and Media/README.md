## `alt` у изображений

```html
<img src="chart.png" alt="Продажи выросли на 20% во втором квартале 2026">
```

| Тип картинки | `alt` |
|--------------|-------|
| Информативная | Описание смысла, не «картинка» |
| Ссылка-картинка | Куда ведёт («Открыть галерею») |
| Декоративная | `alt=""` |

Не дублируйте подпись из `<figcaption>` в `alt` — выберите одно место.

## Декоративные картинки

Разделители, фоновые иллюстрации, иконки рядом с текстом:

```html
<img src="divider.svg" alt="">
<p><img src="icon.svg" alt="" aria-hidden="true"> Бесплатная доставка</p>
```

`alt=""` говорит скринридеру: «пропустить». `aria-hidden="true"` на SVG inline — то же для декора.

## Субтитры и дорожки

```html
<video controls>
    <source src="lecture.mp4" type="video/mp4">
    <track kind="subtitles" src="ru.vtt" srclang="ru" label="Русский" default>
    <track kind="captions" src="ru-cc.vtt" srclang="ru" label="Русские субтитры (глухие)">
</video>
```

| `kind` | Назначение |
|--------|------------|
| `subtitles` | Перевод речи (слышащие, другой язык) |
| `captions` | Текст речи + звуки (для глухих и слабослышащих) |
| `descriptions` | Описание визуального для незрячих |

Формат — WebVTT (`.vtt`).

## Видео и аудио

| Правило | Почему |
|---------|--------|
| `controls` или явная кнопка play | Пользователь сам запускает |
| Не `autoplay` со звуком | Раздражает; браузеры блокируют |
| `title` у iframe (YouTube) | Скринридер знает, что встроено |
| Текстовая альтернатива | Ссылка «Скачать» или расшифровка под видео |
