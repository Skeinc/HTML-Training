## Клавиатура (5 мин)

1. Обновите страницу, нажмите **Tab** с самого начала.
2. Есть ли **skip link** к `#main`?
3. Виден ли **фокус** на каждом элементе?
4. Можно ли открыть меню, отправить форму, закрыть диалог **без мыши**?
5. **Esc** закрывает модалки и выпадашки?

Застряли в ловушке или пропустили кнопку — исправьте порядок Tab и `tabindex`.

## Автоматические инструменты

| Инструмент | Как |
|------------|-----|
| **Lighthouse** | Chrome DevTools → Lighthouse → Accessibility |
| **axe DevTools** | Расширение → Scan страницы |

Инструменты ловят ~30–50% проблем. Остальное — ручная проверка.

## Скринридер (10 мин)

| ОС | Скринридер | Включение |
|----|------------|-----------|
| macOS | VoiceOver | Cmd + F5 |
| Windows | NVDA | Бесплатная загрузка с nvaccess.org |

Пройдите заголовками (ротор/список headings): логична ли иерархия `h1`–`h6`? Озвучиваются ли labels у полей?

## Чеклист

- [ ] Один `h1`, без пропусков уровней
- [ ] У всех полей есть `<label>`
- [ ] У информативных `<img>` есть `alt`
- [ ] У iframe есть `title`
- [ ] Контраст текста ≥ 4.5:1
- [ ] Нет `user-scalable=no` в viewport
- [ ] Ошибки форм — текст + не только цвет
- [ ] Видео со смыслом — субтитры или расшифровка

Пример страницы с типичными ошибками и исправлениями — в [`index.html`](index.html) (сравните комментарии в коде).
