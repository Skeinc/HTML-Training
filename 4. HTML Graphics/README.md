# HTML Graphics

Два способа рисовать на странице: **Canvas** (растр, через JS) и **SVG** (вектор, XML в разметке).

← [HTML Forms](../3.%20HTML%20Forms/README.md) · [HTML Media](../5.%20HTML%20Media/README.md)

1. [Canvas](4.1%20HTML%20Canvas/README.md) — пиксели, анимации, игры
2. [SVG](4.2%20HTML%20SVG/README.md) — масштабируемая векторная графика

| | Canvas | SVG |
|---|--------|-----|
| Тип | Растр (пиксели) | Вектор |
| Разметка | `<canvas>` + JavaScript | `<svg>` с элементами XML |
| Масштаб | Может «мыть» | Без потери качества |
| DOM | Нет объектов фигур | Каждая фигура — узел DOM |
| Когда | Игры, графики, обработка видео | Иконки, диаграммы, интерактив |

Для фото и иллюстраций чаще — `<img>` или CSS, не Canvas/SVG.
