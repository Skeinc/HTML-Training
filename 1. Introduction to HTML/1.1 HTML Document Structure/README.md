Минимальный каркас любой страницы: объявление типа документа, метаданные в `<head>` и видимый контент в `<body>`.

## Минимальный пример

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название вкладки</title>
</head>
<body>
    <!-- видимый контент -->
</body>
</html>
```

Полный пример — в [`index.html`](index.html).

## Части документа

| Часть | Назначение |
|-------|------------|
| `<!DOCTYPE html>` | Режим HTML5; одна строка в самом начале файла |
| `<html lang="…">` | Корневой элемент; `lang` помогает браузеру и скринридерам |
| `<head>` | Метаданные: кодировка, заголовок вкладки, CSS, скрипты |
| `<body>` | Всё, что видит пользователь на странице |

`<!DOCTYPE>` не чувствителен к регистру. В HTML5 достаточно короткой формы `<!DOCTYPE html>`.

## Что класть в head

| Элемент | Зачем |
|---------|-------|
| `<meta charset="UTF-8">` | Кодировка символов (кириллица, эмодзи) |
| `<meta name="viewport" …>` | Корректный масштаб на мобильных |
| `<title>` | Текст на вкладке браузера и в результатах поиска |
| `<link rel="stylesheet" …>` | Подключение внешнего CSS |
| `<script src="…" defer>` | Подключение JavaScript |

`<head>` не отображается на странице — только задаёт окружение для `<body>`.
