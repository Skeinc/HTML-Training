## Что такое HTML?

**HTML** (HyperText Markup Language) — язык **разметки**, не программирования. Он описывает **структуру и смысл** контента: заголовки, абзацы, ссылки, изображения, формы.

| | HTML | CSS | JavaScript |
|---|------|-----|------------|
| **Задача** | Структура и семантика | Оформление | Поведение |
| **Пример** | `<h1>`, `<p>`, `<form>` | цвет, сетка, шрифты | клики, запросы, логика |

Браузер парсит HTML в **DOM**, применяет CSS и выполняет JS. Как страница попадает в браузер по сети — в серии [Интернет](../../Internet/1.%20How%20does%20the%20internet%20work%3F/README.md).
